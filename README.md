# Remax Wechat With TypeScript

使用 Remax 开发微信小程序。

## Getting Start

安装依赖

```bash
npm install
```

调试项目

```bash
# 执行调试命令
$ npm run dev
or
yarn dev
```

使用小程序开发者工具打开项目下的 `dist` 目录

## 构建

```bash
# 执行构建命令
$ npm run build
or
$ yarn build
```

使用小程序开发者工具打开项目下的 `dist` 目录，上传代码即可。
